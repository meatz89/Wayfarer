# Game Design Reference Documentation

This directory contains **detailed reference documentation** for Wayfarer's game design. For the primary Game Design Document (GDD), see [gdd/](../gdd/).

## GDD vs Reference Documentation

| Folder | Purpose | Volume | Read When |
|--------|---------|--------|-----------|
| **gdd/** | Vision, pillars, WHY | ~1,200 lines | Understanding design philosophy |
| **design/** | Details, HOW, exhaustive | ~15,000 lines | Implementing specific systems |

**Start with [gdd/](../gdd/) for orientation**, then reference these documents for depth.
