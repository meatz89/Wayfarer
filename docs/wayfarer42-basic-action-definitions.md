# Wayfarer Location Actions: Basic Definitions

I'll define actions for common locations with clear specifications on card types and interaction patterns. These definitions carefully integrate with the established card system while maintaining elegance and universal applicability.

## Tavern Location: Common Room

### "Approach Innkeeper" (Situation)
**Description:** Interact with the proprietor of the establishment.
**Card Types:** Any (Physical, Intellectual, or Social)
**Approaches Available:**

When using a **Physical card**:
- **Labor Offer:** Requires Strength check (2)
  - *Outcome:* Exchange work for reduced lodging cost or food
  - *Success:* Save 3 coins on room or meal
  - *Benefit in:* Crowded, Interior environments

When using an **Intellectual card**:
- **Information Gathering:** Requires Knowledge check (2)
  - *Outcome:* Learn about the area, available services, current events
  - *Success:* Gain 1-2 rumors or location information
  - *Benefit in:* Quiet, Interior environments

When using a **Social card**:
- **Business Negotiation:** Requires Persuasion check (2)
  - *Outcome:* Negotiate better rates or special services
  - *Success:* 20% discount on services or access to private room
  - *Benefit in:* Crowded, Commerce environments

### "Observe Room" (Intellectual only)
**Description:** Study the tavern's occupants and atmosphere.
**Card Required:** Intellectual
**Approaches Available:**
- **Casual Glance:** No check required
  - *Outcome:* Basic information about patrons present
- **Detailed Observation:** Requires Observation check (2)
  - *Outcome:* Notice relationships between patrons, identify important figures
- **Strategic Analysis:** Requires Analysis check (3)
  - *Outcome:* Identify potential opportunities, conflicts, or valuable contacts

### "Join Evening Activities" (Situation)
**Description:** Participate in the tavern's social entertainment.
**Card Types:** Any (Physical, Intellectual, or Social)
**Approaches Available:**

When using a **Physical card**:
- **Drinking Contest:** Requires Endurance check (3)
  - *Outcome:* Earn respect and possibly coins, risk hangover
  - *Benefit in:* Crowded environments with Bold approach

When using an **Intellectual card**:
- **Game of Strategy:** Requires Planning check (3)
  - *Outcome:* Win coins through games like chess or dice
  - *Benefit in:* Quiet, Interior environments

When using a **Social card**:
- **Share Stories:** Requires Charm check (2)
  - *Outcome:* Build relationships with multiple patrons, gain rumors
  - *Benefit in:* Crowded environments with Bold approach

## Forest Location: Clearing

### "Hunt Game" (Physical only)
**Description:** Seek animals for food and resources.
**Card Required:** Physical
**Approaches Available:**
- **Simple Snare:** No check required
  - *Outcome:* Small chance of catching minor game (1 food)
- **Active Pursuit:** Requires Agility check (3)
  - *Outcome:* Chase and catch medium game (2-3 food)
- **Patient Stalking:** Requires Precision check (3)
  - *Outcome:* Take down larger game (3-4 food, 1 hide)

### "Scan Surroundings" (Intellectual only)
**Description:** Examine the forest environment for resources and information.
**Card Required:** Intellectual
**Approaches Available:**
- **Basic Survey:** No check required
  - *Outcome:* Identify obvious features and common plants
- **Thorough Investigation:** Requires Observation check (2)
  - *Outcome:* Find hidden resources, tracks, or signs of habitation
- **Environmental Analysis:** Requires Knowledge check (3)
  - *Outcome:* Identify rare resources, understand wildlife patterns

### "Navigate Forest" (Situation)
**Description:** Move through the forest toward a destination.
**Card Types:** Any (Physical, Intellectual, or Social)
**Approaches Available:**

When using a **Physical card**:
- **Force Through:** Requires Strength check (2)
  - *Outcome:* Direct path, faster travel, risk of injury
  - *Benefit in:* Outdoor environments with Bold approach

When using an **Intellectual card**:
- **Plot Efficient Course:** Requires Planning check (2)
  - *Outcome:* Safer route, discover resources along the way
  - *Benefit in:* Outdoor environments with Cautious approach

When using a **Social card**:
- **Follow Known Paths:** Requires Knowledge check (2)
  - *Outcome:* Chance of encountering others, gathering information
  - *Benefit in:* Travel environments with Subtle approach

## Town Market: Marketplace

### "Browse Goods" (Situation)
**Description:** Examine merchandise available from various vendors.
**Card Types:** Any (Physical, Intellectual, or Social)
**Approaches Available:**

When using a **Physical card**:
- **Handle Merchandise:** Requires Precision check (2)
  - *Outcome:* Assess quality through direct examination
  - *Benefit in:* Crowded, Outdoor environments

When using an **Intellectual card**:
- **Compare Offerings:** Requires Analysis check (2)
  - *Outcome:* Find best prices, identify valuable items
  - *Benefit in:* Commerce environments with Thorough approach

When using a **Social card**:
- **Chat with Vendors:** Requires Charm check (2)
  - *Outcome:* Learn about merchandise sources, special items
  - *Benefit in:* Crowded, Commerce environments

### "Negotiate Purchase" (Situation)
**Description:** Haggle with merchants over prices.
**Card Types:** Any (Physical, Intellectual, or Social)
**Approaches Available:**

When using a **Physical card**:
- **Impressive Presence:** Requires Strength check (2)
  - *Outcome:* Intimidate for better prices on physical goods
  - *Benefit in:* Crowded environments with Bold approach

When using an **Intellectual card**:
- **Market Knowledge:** Requires Knowledge check (3)
  - *Outcome:* Quote typical prices for fair bargaining
  - *Benefit in:* Commerce environments with Thorough approach

When using a **Social card**:
- **Friendly Haggling:** Requires Persuasion check (3)
  - *Outcome:* Build rapport for discounts or extras
  - *Benefit in:* Commerce environments with Swift approach

## Card Refresh Actions

To complete the system cycle, here are the actions that restore exhausted cards:

### "Rest at Inn" (Physical Refresh)
**Description:** Take proper rest in accommodations.
**Cost:** 3-5 coins depending on quality
**Outcome:** Refresh 1-3 Physical cards (based on accommodation quality)
**Benefit:** Highest effectiveness in locations with Recovery tag

### "Quiet Contemplation" (Intellectual Refresh)
**Description:** Spend time in thought, reading, or study.
**Cost:** 1 hour of game time, requires quiet environment
**Outcome:** Refresh 1-2 Intellectual cards
**Benefit:** Highest effectiveness in locations with Information tag and Quiet property

### "Social Gathering" (Social Refresh)
**Description:** Engage in casual conversation and socializing.
**Cost:** 1 hour of game time, minor coin cost for drinks
**Outcome:** Refresh 1-2 Social cards
**Benefit:** Highest effectiveness in locations with Crowded property

This action system creates meaningful choices across diverse locations while maintaining elegant integration with the card system. The approach selection varies based on card type, but all cards of the same type remain universally applicable across different situations. The system creates natural constraints and opportunities without requiring complex special-case rules or combinations.
