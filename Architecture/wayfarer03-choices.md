# Wayfarer Choices: Complete Reference

## Introduction

Choices in Wayfarer function like abilities or skills in an RPG system. Each choice modifies approach and focus tags while providing mechanical effects on momentum and pressure. This document details the core choices available in the game, organized by approach and effect type.

Each choice includes:
- Tag effects (which approach and focus tags are modified)
- Mechanical effects (momentum increase or pressure decrease)
- Thematic descriptions for different encounter types

## Implementation Notes

### Choice Selection
- Player archetypes initially have access to their preferred approach's basic choices
- Additional choices unlock through character development and experience
- Narrative tags may temporarily block certain focus choices from appearing

### Narrative Integration
- Each choice should be presented with context-appropriate descriptions based on the encounter type
- The Narrative AI Layer transforms the mechanical choices into narratively engaging text

### Strategic Implications
- Players should select choices not just for their immediate effects, but for how they build favorable approach tags
- The value of each choice depends heavily on the strategic tags active in the current location
- Building balanced tag profiles allows greater adaptability across different encounter types
